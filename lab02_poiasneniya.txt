Строка 9: пустая строка.

Строка 12: пустая строка.

Строка 18: if (x <= 0.0) — первая ветка: для x, не превышающего нуля, берётся f1.

Строка 26: ln²x + x; в C++ натуральный логарифм обозначается log.
Строка 27: else — третья ветка: x > a, используется f3.
Строка 28–29: tg²x + x; в C++ тангенс — tan.
Строка 30: закрывающая скобка цепочки if.
Строка 31: пустая строка.
Строка 32: вывод результата y.
Строка 33: return 0 — признак успешного завершения программы.
Строка 34: конец функции main.
