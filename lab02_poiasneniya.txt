ЛАБОРАТОРНАЯ РАБОТА 2 (вариант 25). Пояснение к программе lab02.cpp
====================================================================

НАЗНАЧЕНИЕ
  Программа вычисляет значение кусочно заданной функции y = f(x): на разных
  участках оси x формула разная. Сначала вводятся параметр a и точка x, затем
  по правилу ветвления выбирается одна из трёх формул.

КАК РАБОТАЕТ КОД
1) Считываются a и x (вещественные числа).
2) Условия проверяются в порядке:
   - если x ≤ 0, используется f1: y = sin²x + cos⁴x
     (sin и cos в степени записаны через отдельные переменные sx, cx, чтобы
     не вычислять лишний раз);
   - иначе, если 0 < x ≤ a, используется f2: y = (ln x)² + x
     (логарифм существует только при x > 0, в этой ветке это гарантируется);
   - иначе (x > a) используется f3: y = tg²x + x.
3) Результат y выводится в консоль.

  «Кусочная функция» — это не одна длинная формула, а if / else if / else: для
  каждого участка x своё выражение. Порядок сравнений x с границами 0 и a
  важен: сначала проверяем левый кусок (x ≤ 0), потом средний, потом правый.
