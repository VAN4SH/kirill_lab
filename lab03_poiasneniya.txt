ЛАБОРАТОРНАЯ РАБОТА 3 (вариант 25). Пояснение к программе lab03.cpp
====================================================================

НАЗНАЧЕНИЕ
  На отрезке [xn; xk] с шагом xh строится таблица значений y(x) кусочной
  функции. Используется цикл while. Параллельно ищутся минимальное и максимальное
  значения y на этом отрезке.

КАК УСТРОЕНА ФУНКЦИЯ y (функция y_fun)
  Порог a задан в main как константа (7.0). Для точки x:
  - при x ≤ 0: f1(x) = ctg(x³) + 1, в коде: 1/tan(x³) + 1;
  - при 0 < x ≤ a: f2(x) = (ln(sin x + 1))² — внутри проверяется, что
    sin x + 1 > 0; если слишком близко к нулю, берётся 0, чтобы не «ломать» логарифм;
  - при x > a: f2 заменяется на f3(x) = 2x² + x⁴ + 1.

ТАБУЛИРОВАНИЕ
  x начинается с xn = -7.4, шаг xh = 0.16, цикл while продолжается, пока
  x ≤ xk (к концу отрезка прибавляется маленькое число 1e-9, чтобы не потерять
  последнюю точку из-за погрешностей вещественной арифметики).

ПОИСК ЭКСТРЕМУМОВ
  Первой точке сопоставляются начальные ymin и ymax. Каждое новое y сравнивается
  с ними: если y меньше текущего минимума — обновляем ymin, если больше
  максимума — ymax. В конце выводятся min y и max y на отрезке (это и есть
  «экстремумы в смысле максимума/минимума по сетке», не дифференцирование).

ВЫВОД
  setfixed и setprecision(5) задают вид чисел в консоли (5 знаков после запятой).
