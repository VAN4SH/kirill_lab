ЛАБОРАТОРНАЯ РАБОТА 4 (вариант 25). Пояснение к программе lab04.cpp
====================================================================

НАЗНАЧЕНИЕ
  Задана кусочная функция из трёх ветвей (по варианту, границы: x ≤ 0; 0 < x ≤ 5;
  x > 5). Программа считает y на отрезке [xn; xk] с шагом xh, печатает таблицу
  (для ориентира: три «куска» в WinForms рисуют тремя цветами / тремя сериями).
  Копия данных пишется в текстовый файл lab04_graph.txt — по парам x y, чтобы
  можно было импортировать в Chart или в другой инструмент.

ФОРМУЛЫ
  f1(x) = x / tg x;
  f2(x) = ln(x² + 1);
  f3(x) = e^(-2x) - 3x + 1.

  Функция y_piece(x) возвращает нужное значение: сначала сравнение с 0, потом
  с 5, иначе третья ветвь.

ЦИКЛ
  Используется for по x от xn до xk; к xk снова добавляется 1e-9 из-за
  погрешностей double.

СТРОКА ВЫВОДА
  ofstream out("lab04_graph.txt") — запись; после цикла файл закрывается.

  «Три графика на одном» означает одна ось, три разных формулы на трёх
  отрезках: программа готовит точки, а визуализация в методичке — в WinForms.
