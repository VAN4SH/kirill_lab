# Шпаргалка по ЭА (1 лист А4, супер-кратко)

## 1) База: что такое ЭА
- **Электрический аппарат (ЭА)** — устройство для управления током в электрической цепи.
- Функции: **коммутация**, **защита**, **регулирование**, **изменение параметров U/I**.

## 2) Классификация ЭА (минимум для ответа)
- По назначению: выключатели, разъединители, отделители, автоматы, предохранители, пускатели, реле, регуляторы.
- По области применения: аппараты распределительных устройств, аппараты управления.
- По роду тока: **AC / DC**.
- По уровню напряжения: **высоковольтные / низковольтные**.
- По быстродействию: обычные / быстродействующие.

## 3) Магнитное поле и законы (формулы-ядро)
- **Магнитная индукция**: \(B\), Тл.
- **Напряженность поля**: \(H\), А/м.
- \(\mu_0 = 4\pi \cdot 10^{-7}\ \text{Гн/м}\), \(\mu_r\) — относительная проницаемость.
- **Закон полного тока (Ампера)**:
  \[
  \oint \vec{H}\cdot d\vec{l} = \sum I_i
  \]
- **Закон электромагнитной индукции (Фарадей)**:
  \[
  \mathcal{E}_i = -\frac{d\Phi}{dt},\quad \Phi = BS\cos\alpha
  \]
- **Правило Ленца**: индуцированный ток противодействует причине изменения потока.

## 4) Магнитные материалы (сказать в 1 минуту)
- Диамагнетики: ослабляют поле (\(\chi<0\)).
- Парамагнетики: слабо усиливают (\(\chi>0\), мало).
- Ферромагнетики: сильно усиливают (Fe, Ni, Co).
- Магнитомягкие: малая коэрцитивная сила, сердечники/трансформаторы.
- Магнитотвёрдые: большая коэрцитивная сила, постоянные магниты.

## 5) Магнитная цепь (2 ключевые связи)
- Магнитное сопротивление:
  \[
  R_m = \frac{l}{\mu\mu_0 S}
  \]
- Магнитодвижущая сила:
  \[
  F = WI
  \]
- Аналогия с электрической цепью:
  \[
  \Phi = \frac{F}{R_m}
  \]

## 6) Электромагнит и условия срабатывания
- Электромагнит: катушка + магнитопровод + якорь.
- Виды: с вращающимся якорем, с прямоходовым якорем.
- Для надежного срабатывания: \(F_{эм}(\delta) > F_{пр}(\delta)\).
- Для отпускания: \(F_{пр} > F_{эм}\) при снятии питания.

## 7) Контакты и дуга (самое важное)
- Контактная система: подвижный + неподвижный контакты.
- Типы контактов: точечные, плоские, линейные.
- Электрическая дуга: высокий ток + высокая температура.
- Гашение дуги: узкие щели, дугогасительная решетка, воздух, масло, элегаз (SF6), вакуум.

## 8) Нагрев и теплоотвод
- Механизмы: теплопроводность, конвекция, излучение.
- Закон Фурье:
  \[
  Q = -\lambda S\frac{dT}{dx}
  \]

## 9) Электродинамические силы
- На проводник:
  \[
  F = BIl\sin\alpha
  \]
- Между параллельными проводниками:
  \[
  F = \frac{\mu_0}{2\pi}\frac{I_1I_2}{d}\,l
  \]
- Опасность: деформация токоведущих частей, проблемы контактов при КЗ.

## 10) Аппараты — 1 строка на каждый
- **Контактор**: частые дистанционные коммутации силовых цепей.
- **Промежуточное реле**: развязка и размножение контактов в цепях управления.
- **Реле тока**: защита от перегрузки/КЗ по порогу тока.
- **Реле напряжения**: защита от перенапряжения и просадки напряжения.
- **Тепловое реле**: защита двигателя от длительной перегрузки.
- **Предохранитель**: плавкая вставка, одноразовая защита от аварийного тока.
- **Автоматический выключатель**: тепловой + электромагнитный расцепители, многоразовый.
- **Разъединитель (ВН)**: создает видимый разрыв, без отключения рабочего тока.
- **Высоковольтный выключатель**: отключает ток нагрузки и токи КЗ.
- **Вакуумный выключатель**: быстрое гашение дуги в вакууме, высокая надежность.

## Блиц-вопросы (зубрежка перед ответом)
1. ЭА — это? → Устройство управления током в цепи.  
2. 4 функции ЭА? → Коммутация, защита, регулирование, изменение параметров.  
3. Закон полного тока? → \(\oint Hdl=\sum I\).  
4. Закон Фарадея? → \(\mathcal{E}=-d\Phi/dt\).  
5. Что дает знак «минус»? → Правило Ленца.  
6. Формула магнитного сопротивления? → \(R_m=l/(\mu\mu_0S)\).  
7. МДС? → \(F=WI\).  
8. Условие срабатывания ЭА? → \(F_{эм}>F_{пр}\).  
9. Основные способы гашения дуги? → Щели, решетка, воздух, масло, SF6, вакуум.  
10. Чем автомат отличается от предохранителя? → Многоразовый, имеет расцепители.
