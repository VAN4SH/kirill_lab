ЛАБОРАТОРНАЯ РАБОТА 5 (вариант 25). Пояснение к программе lab05.cpp
====================================================================

НАЗНАЧЕНИЕ
  Работа с одномерным динамическим массивом целых чисел, заполненным
  «случайно», и три типовых задания из таблицы: суммы и произведение.

ЧТО ДЕЛАЕТ ПРОГРАММА
1) srand + time — старт генератора псевдослучайных чисел (чтобы при каждом
   запуске были разные данные, если time меняется).
2) Вводится размер n, создаётся int* a = new int[n].
3) Элементы от -50 до 70: формула -50 + rand() % 121 даёт 121 вариант значения.
4) Массив печатается (это и есть «сгенерировали — показали»).

ПУНКТ a)
  Сумма всех отрицательных элементов: обход, если a[i] < 0 — добавляем к sum_neg
  (тип long long, чтобы не переполнить сумму на большом n).

ПУНКТ b)
  Сумма элементов, чей индекс кратен 3: i % 3 == 0 означает индексы 0, 3, 6, …
  (в C++ нумерация с нуля, как в задании).

ПУНКТ в)
  Находим минимум mn и максимум mx по всему массиву. Ответ: |mn| * |mx|
  (llabs — модуль для long long).

ПАМЯТЬ
  В конце delete[] a — динамический массив обязан быть освобождён.
