# Windows Forms (C++/CLI) — лабораторные, вариант 25

Решение в стиле методички: графические приложения на **.NET Windows Forms** с использованием **C++/CLI** (сборка в **Microsoft Visual Studio** под **Windows**).

## Состав папки

| Путь | Назначение |
|------|------------|
| `KirillLabsWin.sln` | Файл решения — открывать в Visual Studio. |
| `include/MathVariant25.h` | Общие **inline**-функции для варианта 25 (лабы 3, 4, 8): кусочные формулы совпадают с консольными `lab03.cpp`, `lab04.cpp`, `lab08.cpp` в корне `kirill_lab`. |
| `Lab03Win` … `Lab09Win` | Отдельные проекты (см. таблицу ниже). |

**Проектов нет** для лабы 2 (только ввод/вычисление) и **лабы 7** (Монте‑Карло, консоль; при необходимости WinForms‑версию можно сделать по аналогии с другими формами).

## Соответствие лаб и консоли

| Проект | Консольный аналог | Суть интерфейса (кратко) |
|--------|-------------------|--------------------------|
| **Lab03Win** | `lab03.cpp` | Табулирование кусочной функции, поиск min/max. |
| **Lab04Win** | `lab04.cpp` | Параметры `xn`, `xk`, `xh`, кнопка — **Chart** (три кусочка графика). |
| **Lab05Win** | `lab05.cpp` | Массив, суммы и произведение по заданию. |
| **Lab06Win** | `lab06.cpp` | Матрица, пункты a), b), в) из варианта. |
| **Lab08Win** | `lab08.cpp` | **TabControl**: ввод, таблица (DataGridView), график; меню **Файл → Выход**. |
| **Lab09Win** | `lab09.cpp` | Матрица, вектор B по строкам, сортировка вставками. |

## Требования к среде

1. **ОС:** Windows (Linux / WSL графическое Windows Forms **не** запускают — для проверки логики используйте консольные `lab*.cpp` в родительской папке).
2. **Visual Studio** с рабочей нагрузкой **«Разработка классических приложений на C++»** (MSVC).
3. В компонентах установки — поддержка **C++/CLI** (сборка «смешанного» кода .NET + нативный C++).
4. Платформа решения: **x64** (как в `.sln`). Если нет нужного набора инструментов (например, v143), в свойствах проекта: **Общие → набор инструментов платформы** — выберите доступный (v142 и т.д.).
5. **Lab04Win** и **Lab08Win** используют **System.Windows.Forms.DataVisualization** (элемент **Chart**). Если при сборке не находится сборка: **Свойства проекта → Ссылки** — добавьте ссылку на `System.Windows.Forms.DataVisualization` (GAC, .NET Framework 4.x).

## Сборка и запуск

1. Откройте `KirillLabsWin.sln` в Visual Studio.
2. В **Обозревателе решений** выберите нужный проект (например, `Lab04Win`), щёлкните правой кнопкой → **Назначить запускаемым проектом** (при необходимости).
3. **Сборка → Собрать решение** (или только выбранный проект).
4. **Отладка → Начать без отладки** (Ctrl+F5) или **Начать отладку** (F5).

## Примечания

- **Lab08** в текущем виде реализован с упрощённым меню (выход). Диалоги цвета, шрифта, сохранения файла, как в полной методичке, при желании добавляются через дизайнер форм в Visual Studio.
- Формулы и численная логика согласованы с `include/MathVariant25.h` и с консольными программами в `kirill_lab`.
